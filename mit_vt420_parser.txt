    NEW VT420 PARSER IMPLEMENTATION!!!  FREE!!!

=====================================================


This is an announcement of a new release of a parser for
VT420 terminal emulators.  The source code is written in
ANSI C and is a freeware.

The source code for the parser is available via anonymous
ftp net-dist.mit.edu/pub/dos/for-developers/parser.zip

The code can be used with any platform and has been
successfully compiled and ran on DOS and UNIX machines.

The documentation and a sample application are included
in the archive.

The beta-testing has not produced any bug reports.

If you have any comments, suggestions, or have discovered
a bug or a misfeature, please email to the Dos and Windows
development group at dosdev@mit.edu
