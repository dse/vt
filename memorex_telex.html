  Visara 410 Windows-Based Terminal

Connectivity Choices for Lan or Coax Environments

Product Overview
The MTX Visara Windows-based Terminal brings all of this and more to the desktop. It offers high-performance and functionality through a Media GX processor, multi-session and multi-host access, support for both Microsoft RDP and Citrix ICA protocols, and integrated host emulations. The Visara incorporates the Microsoft Windows CE operating system and combines the ease of management and inherent security of a terminal environment with the application and performance of a desktop PC. It eliminates many PC complexities by returning centralized desktop control to the IT manager. Data processing and storage are exclusive to the server or host, therefore creating a true, centrally controlled, client-server environment.

The Visara enables users to simultaneously connect to multiple host systems. Embedded in the onboard memory, the terminal emulation software provides TN5250E, TN3270E and VT220/100 emulation. The Visara∆s integrated software offers a direct connection to the host system. It actually bypasses the server and eliminates the need for supplementary server-based emulation software. Additionally, the Visara passes full screens directly to the host through TCP/IP connections, which increases desktop performance through the reduction of network traffic.

The Visara allows users the ability to run Windows 3.X, Windows 95/98/NT applications through a centrally managed multi-user Windows NT server. This allows for complete PC functionality. The server establishes the individual sessions and all processing occurs on the server. The related presentation screens are sent remotely to the Visara and the administrator controls desktop access.

The MTX Visara Windows-based Terminal brings high-performance, functionality, and a low total cost of ownership to your organization, while maintaining centralized control of your desktops.
